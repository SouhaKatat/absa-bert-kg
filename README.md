# ABSA BERT + Knowledge Graph + Node2Vec

This project implements Aspect Based Sentiment Analysis using:

- BERT embeddings
- Knowledge Graph construction
- Node2Vec graph embeddings

The code was originally developed in Google Colab and later uploaded to GitHub.

## File

asba_bert_transformers_+_kg+_node2vec.py
Main training and evaluation pipeline.

## Requirements

Python libraries used in the project include:

- torch
- transformers
- pandas
- numpy
- networkx
- node2vec
- gensim
- scikit-learn
