# ABSA with BERT + Knowledge Graph + Node2Vec

This project explores Aspect-Based Sentiment Analysis (ABSA) using a hybrid approach combining:

- BERT contextual embeddings
- Knowledge Graph construction
- Node2Vec graph embeddings

The goal is to integrate contextual language models with graph-based semantic relations between aspect terms.

## Repository Structure

src/        → core Python code  
notebooks/  → experimentation and Colab notebooks  
data/       → dataset instructions  
results/    → evaluation outputs and plots
